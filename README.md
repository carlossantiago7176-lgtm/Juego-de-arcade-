# ⚔️ Juego de Batalla por Turnos en Python

Proyecto realizado en Python utilizando Programación Orientada a Objetos (POO).

El juego permite seleccionar un personaje y enfrentarse contra un enemigo aleatorio en una batalla por turnos.

---

# 📌 Características

✅ Selección de personajes  
✅ Sistema de batalla por turnos  
✅ Ataques y defensa  
✅ Herencia  
✅ Encapsulamiento  
✅ Clases abstractas  
✅ Polimorfismo  
✅ Uso de números aleatorios  
✅ Código comentado y explicado  

---

# 🎮 Personajes Disponibles

## 🛡️ Guerrero
- Vida: 100
- Defensa: 20
- Ataque físico balanceado

## 🔮 Mago
- Vida: 80
- Defensa: 10
- Ataque mágico fuerte

## 🏹 Arquero
- Vida: 90
- Defensa: 15
- Ataque rápido y equilibrado

---

# 🧠 Conceptos de POO Utilizados

| Concepto | Implementación |
|---|---|
| Clases | `Guerrero`, `Mago`, `Arquero` |
| Objetos | Creación de personajes |
| Herencia | Todos heredan de `Personaje` |
| Encapsulamiento | Uso de atributos privados `__vida` |
| Abstracción | Clase abstracta `Personaje` |
| Polimorfismo | Método `atacar()` diferente en cada clase |
